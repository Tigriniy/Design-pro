# Design.pro - Веб-портал студии дизайна интерьеров

## Описание проекта
Веб-приложение для студии дизайна интерьеров "Design.pro" с возможностью создания заявок на дизайн, управления заявками и административной панелью.

## Технологии
- Python 3.x
- Django 4.x
- SQLite
- HTML5/CSS3

## Функционал
### Практика 2.1 (реализовано)
- Регистрация пользователей с валидацией
- Авторизация/выход из системы
- Главная страница
- Админ-панель Django

### Практика 2.2 (в разработке)
- Создание заявок на дизайн
- Просмотр своих заявок
- Удаление заявок

### Практика 2.3 (планируется)
- Управление статусами заявок
- Управление категориями

## Установка и запуск
```bash
# Клонирование репозитория
git clone https://github.com/Tigriniy/Design-pro.git
cd Design-pro

# Установка зависимостей
pip install django

# Применение миграций
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser
# Логин: admin
# Пароль: admin

# Запуск сервера
python manage.py runserver 8089
